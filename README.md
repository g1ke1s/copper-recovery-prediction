# Copper Recovery Rate Prediction (XGBoost)

This repository contains the **final ML pipeline** for predicting copper recovery rates using **XGBoost** with **fixed Optuna-tuned hyperparameters**.

---

## Pipeline

- Data loading and cleaning  
- Feature selection and preparation  
- XGBoost training (hyperparameters predefined)  
- Model evaluation (RMSE / MAE / R²)  
- Model export for inference  

---

## Stack

Python · Pandas · scikit-learn · XGBoost · Optuna
